# Notch Technology

متجر إلكتروني PHP بنمط MVC خفيف يدعم واجهة متجر عربية، لوحة تحكم، إدارة المنتجات والتصنيفات والطلبات والعملاء، الدفع عبر Fawateerk، والسلة والمفضلة.

## المتطلبات

- PHP 8.1 أو أحدث.
- MySQL / MariaDB.
- تفعيل امتداد `pdo_mysql`.
- صلاحيات كتابة على مجلدات `uploads/` و `storage/logs/`.

## الإعداد السريع

1. ارفع ملفات المشروع إلى مجلد الاستضافة أو شغّلها محليًا.
2. أنشئ قاعدة بيانات MySQL فارغة.
3. استورد ملف `database/schema.sql`.
4. انسخ `.env.example` إلى `.env` وعدّل قيم الاتصال والدومين، أو اضبط نفس القيم كمتغيرات بيئة على الخادم.
5. افتح الموقع من المتصفح، ثم ادخل لوحة التحكم من `/admin`.

## متغيرات البيئة المدعومة

يمكن ضبط القيم الحساسة والمتغيرة بدون تعديل الكود مباشرة. يقرأ التطبيق القيم من متغيرات البيئة أولًا، ويدعم أيضًا ملف `.env` في جذر المشروع:

| المتغير | الافتراضي | الوصف |
| --- | --- | --- |
| `DB_HOST` | `localhost` | عنوان خادم قاعدة البيانات. |
| `DB_NAME` | `notchtech` | اسم قاعدة البيانات. |
| `DB_USER` | `root` | اسم مستخدم قاعدة البيانات. |
| `DB_PASS` | فارغ | كلمة مرور قاعدة البيانات. |
| `APP_URL` | `https://yourdomain.com` | رابط الموقع الأساسي. |
| `APP_ENV` | `production` | بيئة التشغيل. |
| `APP_DEBUG` | `false` | إظهار أخطاء التطوير عند `true`. |
| `APP_TIMEZONE` | `Africa/Cairo` | المنطقة الزمنية. |
| `APP_CURRENCY` | `EGP` | كود العملة. |
| `APP_CURRENCY_SYMBOL` | `ج.م` | رمز العملة المعروض. |
| `ADMIN_PREFIX` | `admin` | مسار لوحة التحكم. |
| `FAWATEERK_API_KEY` | فارغ | مفتاح Fawateerk. |

## ملاحظات تشغيل

- لا ترفع ملف `.env` إلى Git لأنه يحتوي عادةً على كلمات مرور ومفاتيح دفع.
- اترك `APP_DEBUG=false` في بيئة الإنتاج.
- إذا غيّرت `ADMIN_PREFIX` فسيصبح رابط لوحة التحكم `/<ADMIN_PREFIX>`.
